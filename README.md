# Url_Shortner

Built with React, MongoDB,Express.

Try it out:

Step 1:
`git clone https://github.com/Anandprabhu530/Url_Shortner.git`

Step 2:
```cd client```
```npm install```
```cd ../server```
```npm install```

Step 3:
In index.js replace the variables with your MongoDB credentials.

Step 4:
```cd client```
```npm run dev```

In another new terminal
```cd server```
```npm run start```
